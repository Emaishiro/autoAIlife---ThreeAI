# ThreeAI - AI 角色同居模拟器

一款基于 AI 驱动的同居生活模拟器，支持自定义角色、房间和剧情，通过接入 AI API 实现动态角色对话与行为模拟。

## 功能特点

- **自定义角色**：支持 1-5 个角色，可自定义名字、颜色、性格、职业、技能等属性
- **动态 AI 对话**：接入 DeepSeek / Claude 等 AI API，角色根据性格和场景动态生成回复
- **生活模拟**：时间流逝、日常行为、角色关系度系统
- **存档管理**：3 个本地存档槽位，支持导入导出
- **配置编辑器**：游戏内实时编辑角色和房间配置

## 快速开始

1. 克隆或下载本仓库
2. 用浏览器打开 `index.html`（无需服务器）
3. 在设置面板中填入你的 AI API Key
4. 选择预设或自定义角色，开始游戏

## 支持的 AI API

- DeepSeek API
