# 🎮 读取存档和导入数据功能说明

## 新增功能概述

游戏前设置面板现在新增了 **"读取存档"** 标签页，支持：

1. ✅ **读取本地存档** - 从 3 个存档槽位中快速读取
2. ✅ **导入配置文件** - 从导出的文件导入配置
3. ✅ **自动配置恢复** - 保存的配置自动加载到编辑器中

---

## 📋 读取存档功能

### 使用流程

1. **打开游戏** → 看到设置面板
2. **点击"💾 读取存档"** 标签页
3. **查看存档列表** → 显示 3 个存档槽位

### 存档信息显示

每个存档槽位显示：
- **存档编号** - 存档 1、存档 2、存档 3
- **保存时间** - 何时保存的（日期+时间）
- **配置信息**：
  - 角色数量
  - 房间数量
  - 游戏天数

### 存档操作

#### 读取存档
点击 **"读取此存档"** 按钮：
- ✅ 配置信息自动加载到编辑器
- ✅ 切换到"预设"标签页显示已加载
- ✅ 可继续编辑或直接开始游戏

#### 删除存档
点击 **"删除"** 按钮：
- ⚠️ 确认删除提示
- ✅ 删除后槽位变为空

### 空存档槽位

显示 **[空]**，可以：
- 读取其他存档后在这个槽位保存
- 删除按钮禁用（无需操作）

---

## 📂 导入数据功能

### 支持的文件格式

| 格式 | 说明 | 来源 |
|------|------|------|
| **JSON** | 纯 JSON 格式配置文件 | 导出的配置文件 |
| **TXT** | 文本格式（包含嵌入JSON） | 游戏导出的存档文本版 |
| **混合** | 包含多个存档的文本文件 | 批量导出文件 |

### 导入流程

#### 第一步：选择文件
1. 点击 **"📂 选择文件导入"** 按钮
2. 选择一个本地文件（.json 或 .txt）
3. 文件自动验证并显示预览

#### 第二步：预览内容
导入预览区域显示：
- 📋 **配置快照** 或 **游戏状态** 信息
- 📊 **角色信息** 和 **房间数量**
- 👥 **角色列表** 详细信息

示例预览：
```
📋 配置快照
  角色: 5 个
  房间: 10 个

👥 角色列表
  • 惠舞 (AI工程师)
  • 三玖 (插画师)
  • 五月 (美食博主)
  • Alice (医生)
  • Bob (画家)
```

#### 第三步：确认导入
点击 **"确认导入此配置"** 按钮：
- ✅ 配置加载到编辑器
- ✅ 提示 "配置导入成功"
- ✅ 可继续编辑或直接开始游戏

### 导入错误处理

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| "无法识别文件格式" | 文件不是JSON或TEXT | 检查文件是否为有效格式 |
| "存档数据损坏" | JSON 解析失败 | 手动修复JSON或重新导出 |
| "导入失败: ..." | 配置数据不完整 | 检查导出的文件完整性 |

## 💡 常见用途

### 场景 1：快速切换存档配置

```
流程：
1. 之前保存了"高难度配置"到存档1
2. 又保存了"轻松模式配置"到存档2
3. 现在想切换到存档1的配置
   → 点击"读取存档" → 选择存档1 → 读取
   → 配置自动加载
```

### 场景 2：导入社区配置

```
流程：
1. 从网络下载了朋友分享的配置文件 config.json
2. 打开游戏 → "读取存档" → "选择文件导入"
3. 选择 config.json
4. 预览确认 → 导入
5. 现在可以用这个配置开始游戏
```

### 场景 3：备份和恢复

```
流程：
1. 游戏中到达某个关键点，点击"导出全部存档"
2. 保存导出的 JSON 文件到云盘或U盘
3. 换电脑或重装后，打开游戏
4. "读取存档" → 导入之前导出的文件
5. 所有配置和进度恢复
```

### 场景 4：配置微调

```
流程：
1. 读取某个存档的配置
2. 在"角色配置"标签页调整参数
3. 在"房间配置"标签页修改房间
4. 满意后点击"开始游戏"
5. 新配置自动保存到存档
```

## ⚙️ 技术细节

### 存档格式兼容性

**新格式（v2.0）**：包含 `configSnapshot` 快照
```json
{
  "version": "2.0",
  "configSnapshot": { /* 完整配置 */ },
  "gameState": { /* 游戏状态 */ },
  "saveTimestamp": "2026-03-06 14:30:00"
}
```

**旧格式（v1.0）**：仅包含 `gameState`
```json
{
  "gameState": { /* 游戏状态 */ },
  "saveTimestamp": "..."
}
```

系统**自动兼容**两种格式！旧存档也能导入。

### 自动格式识别

- ✅ 自动检测 JSON 格式
- ✅ 自动从文本中提取 JSON
- ✅ 自动识别 configSnapshot 或 gameState
- ✅ 自动恢复角色信息到编辑器

## 🛠️ 配置编辑后保存

导入配置后，如果进行了编辑：

1. **角色编辑** → 编辑器中修改角色信息
2. **房间编辑** → 编辑器中修改房间信息
3. **点击"开始游戏"** → 新配置自动使用
4. **游戏进行中保存** → 会覆盖原来的存档

## 🚀 使用建议

### ✅ 推荐做法

1. **定期保存** - 每次重要配置都保存到不同槽位
2. **备份导出** - 重要配置导出到本地文件备份
3. **版本管理** - 给导出文件添加版本号或日期
4. **分类管理** - 为不同配置创建不同文件名

### ❌ 避免的做法

1. ❌ 不要删除正在使用的存档
2. ❌ 不要修改导出的 JSON 文件（可能出错）
3. ❌ 不要重复导入相同文件
4. ❌ 不要在游戏中途尝试导入（需先读取存档）
