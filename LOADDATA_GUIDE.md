# 存档与数据导入指南

## 存档槽位（本地读取）

游戏支持 3 个本地存档槽位，存储在 `localStorage`（键名：`apartment_sim_save_1/2/3`）。

### 游戏前读取存档

1. 打开游戏 → 设置面板 → **💾 读取存档** 标签页
2. 每个槽位显示：保存时间 / 角色数 / 房间数 / 游戏天数
3. 点击 **读取此存档** → 配置自动加载到编辑器，可继续修改后开始游戏
4. 点击 **删除** → 弹出确认后清空该槽位

### 游戏中保存 / 读取

右侧控制面板 → **游戏存档** 区域，三个槽位各有 **保存** / **读取** 按钮。

---

## 导出存档

控制面板 → **导出全部存档** 按钮，将所有非空槽位导出为单个 JSON 文件：

```
文件名：apartment_sim_saves_2026-05-06T12-30-00.json
```

**导出格式（多存档汇总）**：
```json
{
  "apartment_sim_save_1": {
    "version": "2.0",
    "configSnapshot": { ... },
    "gameState": { ... },
    "saveTimestamp": "2026-05-06 12:30:00"
  },
  "apartment_sim_save_2": { ... },
  "apartment_sim_save_3": { ... }
}
```

只有非空槽位会出现在导出文件中。

## 导入存档

### 支持的格式

| 格式 | 描述 | 典型来源 |
|------|------|---------|
| **多存档汇总** | 顶层键为 `apartment_sim_save_*` | 游戏"导出全部存档"生成的文件 |
| **单个存档** | 顶层包含 `configSnapshot` 或 `gameState` | 手动编辑或早期导出的单存档文件 |

两种格式均接受 `.json` 或 `.txt` 扩展名。

### 游戏前导入

设置面板 → **💾 读取存档** → **📂 选择文件导入**：

1. 选择文件后自动解析并显示预览（角色列表、房间数、游戏时间）
2. 点击 **确认导入** 后：
   - **多存档汇总**：所有存档分别写入对应槽位（`save_1/2/3`），第一个存档加载到编辑器
   - **单个存档**：写入槽位 1，加载到编辑器
3. 存档列表自动刷新，可直接读取任意槽位继续游戏

### 游戏中导入

控制面板底部 → **导入存档文件**，流程同上，导入后可直接读取。

---

## 存档格式兼容性

| 版本 | 特征 | 处理方式 |
|------|------|---------|
| **v2.0** | 含 `configSnapshot` + `gameState` | 直接读取，完整恢复配置和进度 |
| **v1.0** | 仅含 `gameState` | 从 gameState 重建角色/房间配置，数据可能不完整 |

v1.0 存档读取后建议在编辑器中检查角色/房间信息是否完整。

## 数据流

```
本地 localStorage
    ↓ 读取槽位
配置 + 游戏状态加载到 setupPanelState
    ↓
设置面板各标签页（可继续编辑）
    ↓
点击"开始游戏"
    ↓
activeConfig ← setupPanelState.currentConfig
gameState ← initGameStateFromConfig() + loadedGameState（如有）
    ↓
游戏启动

─────────────────────────────

导入文件 (.json / .txt)
    ↓
解析 → 识别格式（多存档 or 单存档）
    ↓
显示预览
    ↓
确认导入 → 写入 localStorage 对应槽位
    ↓
renderSaveSlots() 刷新存档列表
    ↓
第一个存档加载到编辑器（可继续编辑）
```

---

## 常见错误

| 提示 | 原因 | 处理 |
|------|------|------|
| 文件格式错误 | 文件不含有效 JSON / 无法识别结构 | 确认文件是游戏导出的 JSON |
| 存档数据损坏 | JSON 解析失败 | 用文本编辑器检查 JSON 合法性 |
| 没有可导出的存档 | 三个槽位均为空 | 先在游戏中保存至少一个槽位 |

## 快速参考

| 需求 | 路径 |
|------|------|
| 读取存档继续游戏 | 设置面板 → 💾 → 选槽位 → 读取 |
| 导入外部文件 | 设置面板 → 💾 → 选择文件导入 → 预览 → 确认 |
| 导出所有存档 | 游戏控制面板 → 导出全部存档 |
| 删除某个存档 | 设置面板 → 💾 → 对应槽位 → 删除 |
| 跨设备迁移 | 导出 JSON → 新设备导入 |
